# Mushroom Edibility Classifier

## 1. Описание проекта

Проект представляет собой ML-систему для бинарной классификации изображений грибов на два класса:

- `edible` — съедобный гриб
- `non_edible` — несъедобный или потенциально опасный гриб

Система принимает изображение гриба и возвращает предсказанный класс, а также уровень уверенности модели.

## 2. Бизнес-задача

Цель проекта — разработать прототип системы компьютерного зрения, которая может использоваться в образовательном приложении или справочнике грибника для предварительной классификации изображений грибов.

Ключевая бизнес-проблема: пользователю может быть сложно визуально отличить съедобные грибы от несъедобных. ML-модель может помочь как вспомогательный инструмент для демонстрационной классификации изображений.

## 3. Важное ограничение

Проект является учебным и демонстрационным.

Модель не должна использоваться для принятия реальных решений о безопасности употребления грибов в пищу. Даже при высокой уверенности модели результат не гарантирует, что гриб безопасен.

## 4. Тип задачи машинного обучения

Тип задачи: бинарная классификация изображений.

Входные данные:

- изображение гриба в формате `.jpg`, `.jpeg` или `.png`

Выходные данные:

- предсказанный класс: `edible` или `non_edible`
- вероятность / confidence score

## 5. Метрики качества

Для оценки модели будут использоваться следующие метрики:

- Accuracy
- Precision
- Recall
- F1-score
- Confusion Matrix
- ROC-AUC

Особое внимание будет уделяться recall для класса `non_edible`, так как ошибка, при которой несъедобный гриб классифицируется как съедобный, является наиболее опасной.
